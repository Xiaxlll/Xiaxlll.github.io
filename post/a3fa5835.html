注释

变量

常量

作用：用于记录程序中不可更改的数据

C++定义常量两种方式

  1. #define 宏常量： #define 常量名 常量值
    • ==通常在文件上方定义==，表示一个常量
  2. const修饰的变量 const 数据类型 常量名 = 常量值
    • ==通常在变量定义前加关键字const==，修饰该变量为常量，不可修改

关键字

作用：关键字是C++中预先保留的单词（标识符）

标识符命名

不可以是关键字，只能由字母、数字、下划线组成，第一个字符必须为字母或下划线，字母区分大小写。

数据类型

  • 整型

  • sizeof

  • ~~~c++ //sizeof() //短整型short(2) short num1 = 10; cout << "short占用的内存" << sizeof(num1) << endl; //int(4) cout << "int 类型所占内存空间为： " << sizeof(int) << endl; //long(4) cout << "long 类型所占内存空间为： " << sizeof(long) << endl; //long long(8) cout << "long long 类型所占内存空间为： " << sizeof(long long) << endl;
